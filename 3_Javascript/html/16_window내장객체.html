window 객체

    - 브라우저 창 자체를 나타내는 객체

    - window 객체는 자바스크립트의 최상위 객체이며
    DOM, BOM 으로 분류된다.

    DOM(Document Object Model) : document
    BOM(Browser Object Model) : location, history, screen, navigator

    * location : 주소창 관련 기능을 다루는 객체
                - URL 확인 / 변경 / 이동
    * history : 방문 기록 관련 기능을 다루는 객체
                - 앞/뒤로 페이지 이동
    * screen : 모니터 화면 관련 기능을 다루는 객체
              - 모니터의 전체 화면 해상도 조회
    * navigator : 브라우저와 운영 체제에 대한 정보를 제공하는 객체
                - 브라우저/운영체제 정보, 사용 언어, 위치 정보 등
